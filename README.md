#  Color Flipper

A simple and interactive Color Flipper project built using HTML, CSS, and JavaScript.  
This project allows users to change the background color of the webpage using predefined color buttons or generate a random color dynamically.

---

##  Features

- Change background color instantly
- Random color generator
- Smooth UI interactions
- Beginner-friendly project
- Simple and clean design

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
